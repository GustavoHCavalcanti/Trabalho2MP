\doxysubsection{Detailed Description}
Testes unitários para a função contar\+Palavras. 

Este arquivo contém testes para a função {\ttfamily contar\+Palavras}, que conta a ocorrência de cada palavra em um texto fornecido. A função deve lidar corretamente com texto simples, pontuação e diferenças entre maiúsculas e minúsculas.
